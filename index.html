Features

Free 24/7 Radio

One of the main Hori's features is 24/7 Radio. Radio supports only playlists from YouTube and YouTube Music! Use command /radio_settings to setup radio on your server!

Stable music player

Hori like other music bot has basic music player. Use command /play to play video using search or link. Supported platforms: YouTube, YouTube Music

Other useful features

Hori is being actively developed! We are working on adding lots of interesting features. Right now we've implemented these features: /lyrics; /artist
